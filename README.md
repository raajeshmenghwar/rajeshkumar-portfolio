# Rajesh Kumar - Personal Portfolio

This is the code repository for my personal portfolio website, **[raajeshkumar.tech](https://raajeshkumar.tech)**.

It's a modern, responsive website built with **Next.js**, **Tailwind CSS**, and **Framer Motion** to showcase my projects, skills, and professional experience as a Cyber Security Analyst.

## üèÉ Getting Started

To run this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/raajeshmenghwar/rajeshkumar-portfolio.git](https://github.com/raajeshmenghwar/rajeshkumar-portfolio.git)
    cd rajeshkumar-portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Open your browser:**
    Open [http://localhost:3000](http://localhost:3000) to see the result.
